
## 嵌套
1. 嵌套
2. 嵌套中父选择器标识符&
3. 嵌套中可以使用css选择器
4. 嵌套属性
```scss
.nav{
  padding: 0;
  .nav-item:{
    padding: 15px;
  }
}

// 编译结果
.nav{
  padding: 0;
}
.nav .nav-item:{
  padding: 15px;
}


```

## 父选择器标识符 &
```scss
// === 父选择器标识符 &
article a {
  color: blue;
  &:hover { color: red }
}
// 编译结果
article a { color: blue }
article a:hover { color: red }
```

## 嵌套中的使用选择器
```scss
.container {
  h1, h2, h3 {margin-bottom: .8em}
}
// 编译结果
.container h1, .container h2, .container h3 { margin-bottom: .8em }
```

## 嵌套的css属性
```scss
font: {
  family: Helvetica, sans-serif;
  size: 18px;
  weight: bold;
}

text: {
  align: center;
  transform: lowercase;
  overflow: hidden;
}

// 编译结果
font-family: Helvetica, sans-serif;
font-size: 18px;
font-weight: bold;

text-align: center;
text-transform: lowercase;
text-overflow: hidden;
```
