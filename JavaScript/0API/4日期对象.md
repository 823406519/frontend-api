## 创建对象
```js
new Date();
new Date(value);
new Date(dateString);
new Date(year, monthIndex [, day [, hours [, minutes [, seconds [, milliseconds]]]]]);

// 举例
new Date('2019/05/13')
new Date('2019-05-13')
new Date('2019,05,13,0, 0, 0')
new Date(millseconds) // 1970 Jan 1 开始计算
```
