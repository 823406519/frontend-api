
## 位移
tranlate 在水平和垂直位置位移元素
```css
/* tx 是水平位置上的坐标，ty是垂直位置上的坐标 */
/* 一个长度值或百分比表示X轴和Y轴使用一样的值进行二维上的平移 */
/* 两个长度值或百分比表示在二维上分别按照指定X轴和Y轴的值进行的平移 */
/* 三个长度值或百分比表示在三维上分别按照指定X轴、Y轴、Z轴的值进行的平移 */
transform: translate(tx, ty, tz);


/* translateX()函数表示在二维平面上水平方向移动元素 */
/* translateY()函数表示在二维平面上垂直方向移动元素 */
transform: translateX(tx);
transform: translateY(ty);

 /* translateZ(tz) 等同于 translate3d(0, 0, tz) */
transform: traslateZ(tz)

```
