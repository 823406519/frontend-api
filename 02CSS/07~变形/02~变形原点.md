
## 变形原点
变形的原点设置，操作元素的范围也是可以的
```css
/* === transform-origin 可以设置过渡发生的原点，默认是在元素的center位置上 */
/* 一个值 <length>，<percentage>，或 left, center, right, top, bottom关键字 */
transform-origin: center; /* default */

/* 
两个值： 一个必须是<length>，<percentage>，或left, center, right，
另一个必须是<length>，<percentage>，或top, center, bottom关键字
*/
transform-origin: top left;

/* 
三个值：前两个值和只有两个值时的用法相同，第三个值必须是<length>。它始终代表Z轴偏移量
 */
transform-origin: 2px 30% 10px;
```
