## 水平居中布局

```css
/* === 行内元素 */
/* 只对图片、按钮、文字等行内元素起作用 */
text-aign: center;

/* ==== 已知宽度块级元素 */
width: xxx; margin: 0 auto;
/* 或者 */
width: fit-content; margin 0 auto;

/* 绝对定位 */
position: absolute;
width: $width;
left: 50%;
margin-left: -.5*$width;
/* 或者 */
postion: absolute;
width: $width;
left: 0;
right: 0;
margin: 0 auto;

/* === 未知宽度 */
/* 绝对定位+translate */
position: absolute;
left: 50%;
transform: translate(-50%, 0);
/* flex 布局, 设置项目的容器为flex容器 */
display:flex;
justify-content: center;
```
