
## 分栏布局
一般不把column-count & column-width 一起设置
```css
/* column-count CSS属性，描述元素的列数 */
column-count: 3;

/* column-gap 该 CSS 属性用来设置元素列之间的间隔 (gutter) 大小 */
column-gap: 2rem;
/* column-width CSS属性建议一个最佳列宽 */
column-width: 25em;

column-rule:  <'column-rule-width'> || <'column-rule-style'> || <'column-rule-color'>


/* 
column-span 的值被设置为all时，可以让一个元素跨越所有的列。
一个跨越多列的元素被称为spanning element 
*/
column-span: none | all;
```
