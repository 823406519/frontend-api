
## 背景位置

```css
/* === background-size 定义图像的背景大小 */
/* 第一个值是图像的跨度，第二个值是图像的高度 */
/* 如果只有一个值，那么高度被默认为auto */
/* 
特殊地，
conver  为了使图像完全覆盖容器，保持长宽比，覆盖整个容器
contain  为了使图像以完全显示在容器内，如果图像长宽比与容器长宽比不一致，将会出现空隙
 */
background-size: auto auto; /* default, 保持默人长宽比 */
background-size: 120px 80px;
background-size: contain;
```
