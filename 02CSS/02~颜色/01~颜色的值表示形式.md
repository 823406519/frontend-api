
## 颜色的值的表表示
1. 关键字
2. 十六进制
3. rgb() & rgba()
4. hsl() & hsla()

## 关键字
```css
color: transparent;
color: red; /* 140+ 表示文字的单词 */
```

## 十六进制颜色代码
```css
color: #fff; /* 十六进制颜色代码*/  
```

## rgb() & rgba()
```css
/* 
使用rgb函数，三个参数分别是
第一个值是red
第二个值是green
第三个之是blue
*/
color: rgb(255, 255, 255);
/*
使用rgba函数，第四个参数表示颜色的透明度
*/
color: rgba(255, 255, 255, .3);
```

## hsl() & hsla()
```css
/*
使用hsl函数，三个参数分别是
色调 0-359
饱和度 颜色中的灰度含量 0%-100%
亮度 0%-100%  0%是黑色，100%是白色，50%是标准色
*/
color: hsl(14, 100%, 53%);
/*
使用hsla函数表示，第四个参数表色颜色的透明度
*/
color: hsla(14, 100%, 53%, 0.6);
```
