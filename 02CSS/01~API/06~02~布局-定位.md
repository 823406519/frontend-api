
## 定位
```css
/* === 使用position属性进行定位 */
/* default 元素保持在正常文档流
偏移属性和z-index 不能生效
*/
position: static; 

/* 保持在正常文档流，相当与元素原来正常文档流中的位置进行偏移 */
position: relative;
left: 10px; /* 相对与元素原来的位置向左移动10px */

/* 脱离文档流
相当于最近的非static定位的祖先元素进行定位，如果都不指定定位
，则相对于整个html文档
 */
position: absolute;

/* 脱离文档流，相对于浏览器视口进行定位 */
position: fixed;
```
