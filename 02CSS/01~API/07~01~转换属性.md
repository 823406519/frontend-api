
## 转换属性
```css
/* === transform转换属性可以设置元素的 rotate scale skew translate */
transform: translate(120px, 50%);
transform: scale(2, 0.5);
transform: skew(30deg, 20deg);
transform: rotate(0.5turn);

/* === transform-origin 可以设置过渡发生的原点，默认是在元素的center位置上 */
transform-origin: center; /* default */
transform-origin: top left;
transform-origin: 50px 50px;
```

## 转换的各种函数
```css
/* === rotate 设置元素的旋转的变换，如果是正的角度，那么是顺时针进行旋转 */
transform: rotate(<angle>) 
/* 其他旋转函数 */
transform: rotateX(<angle>) 
transform: rotateY(<angle>) 
transform: rotateZ(<angle>) 
transform: rotate3d(x, y, z, <angle>) /* xyz参数旋转轴的矢量在0到1之间 */

/* === scale 设置元素的缩放转换*/
transform: scale(sx)
transform: scale(sx, sy)
transform: scaleX(s)
transform: scaleY(s)
transform: scaleZ(s)

/* ===skew 设置元素的偏斜转换, skew 没有z轴的偏移*/
transform: skew(ax)
transform: skew(ax, ay)
transform: skewX(ax)
transform: skewY(ay)

/* === translate 设置元素的平移转换*/
transform: translate(tx);
transform: translate(tx, ty);
transform: translateX(tx);
transform: translateY(ty);

/* ===  perspective 设置用户与z=0的平面之间的距离*/
transform: perspective(0);
transform: perspective(800px);
```
