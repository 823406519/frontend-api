## display属性
1. block       
  元素会被显示为块级元素，可设置宽高，换行显示             
2. inline       
  元素会被显示为内联元素, 默认高宽为内容高宽，不可设置高宽，同行显示 
3. inine-block  
  元素会被显示为行内块元素, 默认高宽为内容高宽，**可设置高宽**，同行显示
4. list-item      
  元素会作为列表显示                                       
5. table        
  元素会作为块级表格来显示        
6. none     
  元素不会显示，已脱离文档流
7. inherit    
  指从父元素继承 display 属性的值
