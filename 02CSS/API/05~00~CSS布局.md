## position:absolute 与 float的异同
1. 都是脱离文档流，并且可以设置宽高
2. float不会遮盖另一个BFC区域；position:absolute会产生一个遮盖

## div+css 布局较table布局的优点
1. 代码更加结构化和简洁，方便维护，加载速度更快
2. 搜索引擎更友好
