## 超出宽度的文字显示为省略号
```css
selector{
  overflow: hidden;
  width: xxx;
  white-space: nowrap;
  text-overflow: ellipsis;  
}
```

## 英文单词发生词内换行
```css
word-wrap: break-word;
```

##  css 中可以让文字在垂直和水平方向上重叠的两个属性
1. line-height
2. letter-spacing

> letter-spacing还可以用于消除inline-block元素间的换行符空格间隙问题

## 单词间距与字母间距属性
word-spacing : 单词

letter-spacing : 字母、中文
