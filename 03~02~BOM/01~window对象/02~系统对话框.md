
## 系统对话框
1. alert
2. confirm
3. prompt

```js
// === alert 显示一个警告对话框还有ok按钮
alert(message)

// === confirm 显示一个对话框，具有ok&cancel按钮
// 点击ok按钮，返回true；cancel 按钮返回false
confirm(message) 

// === prompt 显示一个对话框，提示用户输入，具有ok&cacel按钮    
// 点击ok按钮 返回输入框中的文本值；点击cancel 返回null
prompt(questionStr[,defaultStr])
```
