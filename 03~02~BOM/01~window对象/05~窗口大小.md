
## 窗口大小
```js
// === 页面视图区大小
window.innerWidth // 获得浏览器窗口的内容区域的宽度，包含垂直滚动条(如果有的话)。
window.innerHeight // 获得浏览器窗口的内容区域的高度，包含水平滚动条(如果有的话)

document.documentElement.clientWicth // 相当于window.innerWidth
document.documentElement.clientHeight // 相当于window.innerHeight

// === 浏览器尺寸
window.outerWidth // 返回浏览器窗口的外部宽度，包括所有的界面元素，如滚动条，工具条
window.outerHeight // 返回浏览器窗口的外部高度，包括所有的界面元素，如滚动条，工具条



// 此属性表示窗口是否以全屏显示。
Window.fullScreen

// === 调整大小
/* 
 一般默认情况下，不允许对浏览器主窗口进行位置移动，位置移动，窗口关闭等操作
 允许对window.open新打开的窗口进行上述操作，window.open可以返回一个指代创建的窗口对象
 */
// 调整到指定的x y 像素大小
window.resizeTo(x, y)

// 接受新窗口与旧窗口x y 的差
window.resizeBy(x, y)
```
