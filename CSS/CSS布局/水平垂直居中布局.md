1. 元素有固定宽度和高度
  ```css
  selector{
    position: absolute;
    top: 50%;
    left: 50%;

    width: $width;
    height: $height;

    /* 注意，移动的是元素的box宽度 */
    marign: -$width/2, 0, 0, -$height/2;
  }
  ```

2. 元素没有固定宽度和高度
  ```css
  selector{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
  }
  ```

3. flex
  ```css
   selector{

    justify-content: center;
    align-items: center;
  }
  ```
