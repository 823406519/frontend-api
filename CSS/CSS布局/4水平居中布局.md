# 水平居中

## 行内元素
```scss
/* 行内元素 */
text-aign: center;
```

## 块级元素
1. 已知宽度
  ```scss
  margin：0 auto;
  ```

2. 有float属性
  ```scss
  {
    width: fit-content;
    margin: 0 auto;
  },
  ```

3. flex 布局
  ```scss
  {
    display: flex;
    justify-content: center;
  }
  ```

4. 绝对定位
   ```scss
   /* 绝对定位未知宽度 */
  {
    position: absolute;
    left: 50%;
    transform: translate(-50%, 0);
  }

   /* 绝对定位已知宽度  letf 50%*/
  {
    position: absolute;
    width: $width;
    left: 50%;
    margin-left: .5*$width;
  }
  {
    /* 绝对定位已知宽度  letf right 0*/
  {
    postion: absolut;
    width: $width;
    left: 0;
    right: 0;
    margin: 0 auto;
  }
  ```
