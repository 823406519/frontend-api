# 垂直居中

## 行内元素
```scss
line-height: $height;
```

## 块级元素
1. table
  ```scss
  // 父
  {
    display: table;
  }
  // 子
  {
    display: table-cell;
    vertical-align: middle;
  }
  ```
2. flex
  ```scss
  {
    display: flex;
    align-items: center;
  }
  ```

3. 绝对定位
 ```scss
 // 未知宽度
  {
    position: absolute;
    top: 50%;
    transform: translate(0, -50%);
  }

  // 已知宽度top 50%
  {
    position: absolute;
    top: 50%;
    height: $height;
    margin-top: .5*$height;
  }

  // 已知宽度top bottom 0
  {
    position: aboulte;
    top: 0;
    bottom: 0;
    margin: auto 0;
  }
  ```
