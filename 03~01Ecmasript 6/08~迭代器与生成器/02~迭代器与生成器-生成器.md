
## 生成器
generator 是一个返回迭代器的函数。

```js
// yield 关键字只能用于生成器函数中，即使是在生成器内部的非生成器函数也是错误的

// * 指示这是一个生成器函数，返回一个迭代器对象
function* createIterator() {
  // yield 指示了在next方法被调用时返回的值
  // 函数调用在yield语句执行后，将停止往下执行，知道迭代器再次调用next方法
  yield 1;
  yield 2;
  yield 3;
}
const iterator = createIterator();
iterator.next(); // {value: 1, done: false}
iterator.next(); // {value: 2, done: false}
iterator.next(); // {value: 3, done: false}
iterator.next(); // {value: undefined, done: true}

// ===== 生成器函数表达式
const iterator = function *crateIterator(){
  // ...
}
// ==== 不能使用箭头函数创建生成器函数

// === 对象中生成器函数的缩写
let obj = {
  *createIterator(){
    // ...
  }
}
```
