
## 模板字面量 template literal
与普通字符串字面量相比：
1. 允许多行字符串
2. 已存在变量值的替换（字符串连接变得更简单）
3. HTML转义，转换字符串使其更加安全地插入到HTML中

```js
// 模板字符串使用`需要使用\进行转义
// 模板字符串中的 ' " 不需要使用\进行转义

let message = `Liu said, "tempalte iteral used surounded with \`"`

// 多行字符串
// 模板字符串前面的空格也会当作字符串的一部分
let message `Multiline
String`;

// 如果缩进很重要，可以这样做
let html = `
<div>
    <p> hello world</p>
</div>`.trim(); // 使用trim移除起始空行

// 使用 ${} 嵌入js表达式
let name = 'liuyaohui';
let sayHello = `hello ${name}`;
sayHello = `hello ${name + '!'}`;
```

## 模板标签
```js
/*
tag 是一个函数，调用时接受模板字面量数据.
第一个参数是被js解释过的字面量
随后的参数是各个替换位表达式的解释值
*/
// literals.length = substitutions.length + 1
let message = tag`hello world`;
function tag(literals, ...substitutions){}

// ==== example
let count = 10;
let price = 25;
tag`count is ${count}, price is ${count * price} yuan!`;

function tag(literals, ...args) {
  console.log(literals); // [ 'count is ', ', price is ', ' yuan!' ]
  console.log(args); // [ 10, 250 ]
}
```

## 模板字面量原始值
模板字面量获取原始字符串值可以使用`String.raw`方法
```js
String.raw`multiline\nstring` // "multiline\nstring"

// ==== 模拟raw
let count = 10;
let price = 25;

String.raw = function(literals, ...args) {
  console.log(literals); // [ 'count is ', ', price is ', ' yuan!' ]
  console.log(literals.raw); // [ 'count is ', '.\\nprice is ', ' yuan!' ]
  console.log(args); // [ 10, 250 ]
};

String.raw`count is ${count}.\nprice is ${count * price} yuan!`;
```
