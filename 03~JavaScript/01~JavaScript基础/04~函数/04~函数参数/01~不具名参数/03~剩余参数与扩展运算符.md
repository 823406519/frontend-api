
## 扩展运算符
剩余参数将多个参数合并到一个数组，而扩展运算符则将数组分解为各个项。

```js
// === 例子
// 对于Math.max 求解数组，ES6前这样解决
let values = [25, 50, 75, 100];
console.log(Math.max.apply(Math, values)); // 100

// ES6可以使用扩展运算符
let values = [25, 50, 75, 100]
// equivalent to
// console.log(Math.max(25, 50, 75, 100));
console.log(Math.max(...values));           // 100
```
