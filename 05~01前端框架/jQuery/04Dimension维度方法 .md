## Dimensions 方法
1. $(selector).width()
2. $(selector).innerWidth() -> width + padding
3. $(selector).outerWidth() -> width + padding + border
4. $(selector).height()
5. $(selector).innerHeight() -> height + padding
6. $(selector).outerHeight() -> height + padding + border

> $(document).width
> $(window).width
