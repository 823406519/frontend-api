
## jQuery 添加节点
```js
$(selector).append(node) // 将node增加到指定的选择器的节点内部最后
$(selector).appendTo(node) // 将指定的选择器的节点增加到node内部的最后


$(selector).prepend(node) // 将node增加到指定的选择器的节点内部的最前
$(selector).prependTo(node) // 将指定的选择器的节点增加到node内部的最前

// 将cotent增加到指定选择器的节点的后面
$(selector).after(content[,function(index)])
$(content).insertAfter(selector) 

// 将cotent增加到指定选择器的节点的前面
$(selector).before(content,function(index))
$(content).insertBefore(selector)
```
