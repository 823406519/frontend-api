
## JQuery 关系节点
```js
/*
区分用法：
1. parent() 用于返回直接父元素
2. parents() 用于返回所有的祖先元素直到根元素
3. parentsUntil() 返回selector与stop之间的祖先元素
*/
$(selector).parent(filter)
$(selector).parents(filter)
$(selector).parentsUntil(stop,filter)

/*
区分用法：
1. childern() 直接子元素
2. find() 下层所有的子元素
*/
$(selector).children(filter)
$(selector).find(filter)


/*
区分用法：
1. prev() 前一个兄弟元素
2. prevAll)() 前面所有的兄弟元素
3. prevUntil() 在selector与stop之间的的所有的兄弟元素

1. next() 前一个兄弟元素
2. nextAll)() 前面所有的兄弟元素
3. nextUntil() 在selector与stop之间的的所有的兄弟元素

1. siblings() 返回所有的兄弟元素
*/
$(selector).prev(filter)
$(selector).prevAll(filter)
$(selector).prevUntil(stop,filter)

$(selector).next(filter)
$(selector).nextAll(filter)
$(selector).nextUntil(stop,filter)

$(selector).siblings(filter)
```
