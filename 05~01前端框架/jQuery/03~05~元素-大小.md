
## jQuery 元素大小和位置
```js
// width, 不包括padding，border，margin
// return width
$(selector).width() // 返回第一个匹配selector的元素的宽度
// set width
$(selector).width(value) // 为所有匹配selector的元素设置宽度
$(selector).width(function(index,currentwidth))

// innerwidth = width + padding
$(selector).innerWidth() 

// outerWidth = width + padding + border
$(selector).outerWidth(isIncludeMargin)

// === 同理，可以得到相应的height
```
