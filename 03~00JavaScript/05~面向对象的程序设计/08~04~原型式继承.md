
## 原型式继承
基于已有的对象，将其作为其他对象的原型对象，以此实现继承，而不必自定义类型
```js
function object(obj){
  function F(){};
  F.prototype = obj;
  return new F();
}

// ===== ECMAScript 5 新增Object.create() 规范原型式继承
// 这个方法接收两个参数：
// 一个用作新对象原型的对象和（可选的）一个为新对象定义额外属性的对象
let obj = {
  foo: 'bar'
}
let anotherObj = Object.create(obj, {
  name: 'liuyaohui'
})
anotherObj.foo; 'bar'
anotherObj.name; 'liuyaohuis'
```
