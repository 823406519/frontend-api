## DOM
Document Object Model

> DOM将整个HTML文档看作是节点树

# DOM节点类型
1. 文档节点(根节点) document
2. 元素节点
3. 属性节点
4. 文本节点

| 节点类型 | nodeType | nodeName | nodeValue |
| -------- | -------- | -------- | --------- |
| 元素节点 | 1        | 元素名称 | null      |
| 属性节点 | 2        | 属性名称 | 属性值    |
| 文本节点 | 3        | #text    | 文本内容  |

JS中所有节点都是对象

## 节点关系
1. 父节点(parent) 子节点(child)
2. 祖先节点（ancestor) 后代节点(descendant)
3. 兄弟节点（sibling)
  * 紧邻兄弟节点 adjacent sibling
