## 元素节点属性
```js
element.tagName // 标签
element.innerHtml

element.attributes  // 返回节点的属性集合，类数组对象
element.attributes.setNamedItem(node)
element.attributes.getNamedItem
element.attributes.removeNamedItem(name)
element.attributes[name] // 获取属性节点

element.attributeName
/*======= example
id, value, style, className
*/
element.getAttribute(attributeName)
element.setAttribute(name, value)
element.removeAttribute(attrName)

// ===== 访问节点样式属性
element.style.xxx  // 只能获取行内样式的属性
window.getComputedStyle(element[, pseudoElt]) // 获取当前样式
// DOM-扩展 classList
element.classList.add(className)
element.classList.remove(className)
element.classList.toggle(className)
element.classList.contains(className)
```
> 区别：`.`操作符不能设置和访问自定义属性
