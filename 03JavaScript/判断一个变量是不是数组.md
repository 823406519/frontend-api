1. instanceof 判断 可以
  ```js
  let a []
  a instanceof Array // true
  ```

2. 用Array对象的isArray方法判断(不同的frame,会判断错误)
  ```js
  let a = []
  Array.isArray(a) // true
  ```

3. constructor
  ```js
  // 不准确,可以指定 obj.constructor = Array
  arr.constructor === Array
  ```

4.  Object.prototype.toString.call,如果值是 [object Array], 说明是数组
