## 修改器方法
修改器方法: 方法会改变调用它们的对象自身的值。

1. 栈与队列方法
2. reverse
3. sort
4. splice

## 栈与队列方法
```js
// 将一个或多个元素添加到数组的末尾，并返回该数组的新长度
Array.prototype.push(element1, ..., elementN)
// 删除数组的第一个元素，并返回这个元素
Array.prototype.shift()

// 在数组的开头增加一个或多个元素，并返回数组的新长度
Array.prtotype.unshift(element1, ..., elementN)
// 删除最后一个元素，并返回该元素的值。此方法更改数组的长度
Array.prototype.pop()
```

## reverse
```js
// 将数组中元素的位置颠倒，并返回颠倒后的数组，注意，原数组同样被修改
Array.prototype.reverse()
```

## sort
```js
// 用原地算法对数组的元素进行排序，并返回数组
// 如果没有指明 compareFunction ，那么元素会按照转换为的字符串的诸个字符的Unicode位点进行排序
// 如果指明了 compareFunction ，那么数组会按照调用该函数的返回值排序
  // 如果 compareFunction(a, b) 小于 0 ，那么 a 会被排列到 b 之前
  // 如果 compareFunction(a, b) 等于 0 ， a 和 b 的相对位置不变
  // 如果 compareFunction(a, b) 大于 0 ， b 会被排列到 a 之前
Array.prototype.sort([compareFunction])

// === 例子
// 要比较数字而非字符串，比较函数可以简单的以 a 减 b，如下的函数将会将数组升序排列
function compareNumbers(a, b) {
  return a - b;
}

// === 例子2
var numbers = [4, 2, 5, 1, 3]; 
numbers.sort((a, b) => a - b); 
console.log(numbers);

// [1, 2, 3, 4, 5]
```

## splice
```js
// 删除或替换现有元素或者添加新的元素来修改数组,返回修改后的数组，此方法会改变原数组。
// start 表示修改的开始位置
// deleteCount 可选的，表示要移除的数组元素的个数
// item1, item2, ...  可选，表示添加的元素列表
Array.prototype.splice(start[, deleteCount[, item1[, item2[, ...]]]])
```
