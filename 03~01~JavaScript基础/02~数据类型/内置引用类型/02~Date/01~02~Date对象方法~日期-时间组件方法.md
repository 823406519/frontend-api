
## 时间组件方法
```js
// 获取从1970/1/1至今的毫秒数，设置毫秒数
Date.prototype.set/getTime 

Date.prototype.set/getFullYear()
//  0-11   ！！！！！！！
Date.prototype.set/getMonth()
// 1-31
Date.prototype.set/getDate() 
// 0-6 ！！！！！！！！！！ 0是周日
Date.prototype.getDay() 
// 0-23
Date.prototype.set/getHours() 
Date.prototype.set/getMinutes() // 0-59
Date.prototype.set/getMilliseconds() 0-59
```
